## Overview

Creates a new bridge session with a unique smart wallet for transferring USDT from one chain to another.

## Request Body

<ParamField body="sourceChain" type="string" required>
  The source blockchain network. Must be either `sepolia` or `base-sepolia`.
</ParamField>

<ParamField body="destinationChain" type="string" required>
  The destination blockchain network. Must be either `sepolia` or `base-sepolia`, and different from sourceChain.
</ParamField>

<ParamField body="amountUSDT" type="string" required>
  Amount of USDT to bridge (in decimal format, e.g., "10" for 10 USDT).
</ParamField>

<ParamField body="destinationWallet" type="string" required>
  The wallet address that will receive USDT on the destination chain. Must be a valid Ethereum address.
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="sessionId" type="string">
  Unique identifier for this bridge session
</ResponseField>

<ResponseField name="depositAddress" type="string">
  Address where you should send USDT on the source chain
</ResponseField>

<ResponseField name="smartWalletAddress" type="string">
  The smart wallet address (same as depositAddress)
</ResponseField>

<ResponseField name="sourceChain" type="string">

<ResponseField name="destinationChain" type="string">

<ResponseField name="amountUSDT" type="string">

</ResponseField>

<ResponseField name="destinationWallet" type="string">

  </Expandable>

## What Happens Next?

After creating a bridge request:

<Steps>
  <Step title="Send USDT">
    Transfer the exact amount of USDT to the deposit address on the source chain
  </Step>
  
  <Step title="Automatic Detection">
    The system monitors the deposit address and detects your USDT transfer (checks every 5 seconds)
  </Step>
  
  <Step title="Auto-Funding">
    The system automatically funds the smart wallet with ETH for gas fees
  </Step>
  
  <Step title="Bridge Execution">
    The bridge transaction is executed automatically:
    - Approves USDT for the bridge contract
    - Calls the bridge function with LayerZero
  </Step>
  
  <Step title="Completion">
    Your USDT arrives on the destination chain at the specified wallet address
  </Step>
</Steps>

<Note>
  The entire process after depositing USDT is **fully automated** - you don't need to do anything else!
</Note>

## Important Notes

<Warning>
  - Send the **exact amount** of USDT specified in the request
  - Send USDT on the **correct source chain**
  - The session will timeout after **20 minutes** if no deposit is detected
  - Each session creates a **unique smart wallet** - don't reuse deposit addresses
</Warning>

```bash
curl https://stable-flow-backend.onrender.com/bridge/status/bridge_1737638400000_abc123def
```

## Fee Information

When bridging USDT, the following fees apply:

- **LayerZero Fee**: Paid in ETH (automatically funded by the system)
- **LP Fee**: Deducted from USDT amount
- **Protocol Fee**: Deducted from USDT amount

The recipient will receive slightly less than the amount sent due to LP and protocol fees.

<Tip>
  Use a test amount first (e.g., 1-10 USDT) to understand the fees before bridging larger amounts.
</Tip>

## Example Complete Flow
```javascript
// Step 1: Create bridge request
const createResponse = await fetch('https://stable-flow-backend.onrender.com/bridge/create', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    sourceChain: 'base-sepolia',
    destinationChain: 'sepolia',
    amountUSDT: '10',
    destinationWallet: '0xYourAddress'
  })
});

const bridgeData = await createResponse.json();
console.log('Session ID:', bridgeData.sessionId);
console.log('Send USDT to:', bridgeData.depositAddress);

// Step 2: Send USDT to depositAddress using your wallet
// (User action required)

// Step 3: Monitor status
async function checkStatus() {
  const statusResponse = await fetch(
    `https://stable-flow-backend.onrender.com/bridge/status/${bridgeData.sessionId}`
  );
  const status = await statusResponse.json();
  console.log('Current status:', status.status);
  
  if (status.status === 'completed') {
    console.log('Bridge completed!');
    console.log('TX Hash:', status.txHash);
    return true;
  }
  return false;
}

// Poll every 10 seconds
const interval = setInterval(async () => {
  const completed = await checkStatus();
  if (completed) {
    clearInterval(interval);
  }
}, 10000);
```
